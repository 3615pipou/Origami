Graphs

INSTRUCTIONS

Graphs are a collection of nodes connected by edges (circles connected by lines).

On their own, nodes and edges store properties of the particular thing they represent, but edges must also always store the 2 nodes that they connect.

Nodes and edges are both stored in arrays, and each edge is an array of 2 numbers: the indices of the 2 nodes (in the node array) it connects.

var graph = new Graph()
console.log(graph)

Get Nodes

INSTRUCTIONS

To get all the nodes, access the .nodes property of the graph. To get certain nodes based on adjacency, use either getNodesAdjacentToNode or getNodesAdjacentToEdge.

"adjacency" has to do with edge-connections and has nothing to do with position in space. Nodes are adjacent when they are connected by an edge.

graph.nodes;  // array
graph.getNodesAdjacentToNode()
graph.getNodesAdjacentToEdge()

Get Edges

INSTRUCTIONS

To get all the edges, access the .edges property of the graph. To get certain edges based on adjacency, use either getEdgesAdjacentToNode or getEdgesAdjacentToEdge.

"adjacency" has to do with edge-connections and has nothing to do with position in space. Two edges are adjacent when they are both connected to the same node.

graph.edges;  // array
graph.getEdgesAdjacentToNode()
graph.getEdgesAdjacentToEdge()
